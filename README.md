# Claude-Like Theme

![Claude-like Theme](image/2026-03-11-claude-like-theme/light.png)

一个适用于 Typora 的 Claude-like 阅读主题。

## 项目来源

本项目 fork 自 [Muyiiiii/Typora_Claude-Like_Theme](https://github.com/Muyiiiii/Typora_Claude-Like_Theme)。

当前仓库在原项目基础上做了精简，仅保留主要主题文件 `claude-like.css`，移除了 dark / grey 相关主题文件与说明。

## 特性

- 保留 Claude-like 风格的浅色阅读主题。
- 增加 Mermaid 图表样式适配：统一序列图中的参与者、连线、标签和注释样式，并同时覆盖 Typora 编辑预览与导出结果。

## 安装

1. 打开 Typora。
2. 进入 `设置 / 偏好设置 -> 外观 -> 打开主题文件夹`。
3. 将 `claude-like.css` 复制到主题目录。
4. 重启 Typora。
5. 在主题菜单中选择 `Claude-like`。

## 文件

- `claude-like.css`：Typora 主题样式
- `image/2026-03-11-claude-like-theme/light.png`：主题预览图
