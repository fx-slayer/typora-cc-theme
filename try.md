# Claude-Like Theme

这是一个适合长文阅读的 Typora 主题。

## 特点

- 排版克制
- 表格统一
- 代码块清晰
- 支持浅色与深色主题

## 示例段落

你可以用它写技术笔记、课程总结、论文阅读记录，也可以整理 AI 对话内容。 
整体风格偏安静、简洁，适合持续阅读。

## 表格示例

| 功能     | 说明   |
| -------- | ------ |
| 标题层级 | 更清晰 |
| 表格样式 | 更统一 |
| 代码高亮 | 更易读 |

## 复杂度分析表

| 操作                           | 时间复杂度 | 空间复杂度 | 说明                                 |
| ------------------------------ | ---------- | ---------- | ------------------------------------ |
| `print()` 函数调用             | O(1)       | O(1)       | 直接输出字符串，不涉及循环或递归     |
| 字符串字面量 `"Hello, World!"` | O(1)       | O(1)       | 字符串长度固定，编译时确定           |
| 整个程序执行                   | O(1)       | O(1)       | 单次顺序执行，无复杂的数据结构或算法 |

## Python代码

```python
"""
Hello World 程序
"""

print("Hello, World!")
```

## 分析说明

- **时间复杂度 O(1)**：`print()` 是一个常数时间操作，输出固定长度的字符串，不管输入数据大小如何都是常数时间
- **空间复杂度 O(1)**：程序不创建任何随输入大小增长的数据结构，只占用固定的栈和堆空间
